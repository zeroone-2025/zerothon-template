# 📢 마케팅 문서

> **Claude Code 활용 안내**: `@docs/01_plan.md`를 참조하면 타겟 사용자와 비즈니스 모델을 자동으로 확인할 수 있습니다. 아래 프롬프트를 참고하여 Claude Code에서 자연어로 요청하세요. 이 문서를 통해 **5가지 산출물**(카드뉴스, 에브리타임 글, AI 쇼츠, 시연+발표 쇼츠, 블로그 포스트)을 완성하고 **4개 SNS 플랫폼**에 배포합니다.
>
> **기획자가 정한 타겟과 핵심 메시지 방향을 기준으로 작업하세요.** 방향이 달라질 것 같으면 기획자와 먼저 맞춰보세요.

## 템플릿별 작성 가이드

### 1. 카드뉴스 4장

- **목적**: 문제→해결 흐름을 시각적으로 전달
- **포맷**: 1080×1080 정사각형 4장
- **작성 가이드**:
  - 1장 — 후킹: 타겟의 고민을 질문이나 공감 문장으로 시작
  - 2장 — 문제 공감: 기존 방식의 불편함을 구체적으로 묘사
  - 3장 — 솔루션: 우리 서비스가 해결하는 방법 (스크린샷 포함)
  - 4장 — CTA: 서비스 링크 + 한 줄 행동 유도

| 장 | 제목 | 본문 카피 | 시각 요소 |
|----|------|-----------|-----------|
| 1장 (후킹) | | | |
| 2장 (공감) | | | |
| 3장 (솔루션) | | | |
| 4장 (CTA) | | | |

> **AI 프롬프트** -- 카드뉴스 구성안 생성
>
> ```
> @docs/01_plan.md 참고해서 인스타그램 카드뉴스 4장 구성안을 만들어줘.
> 슬로건: {{선택한 슬로건}}
> 톤앤매너: {{원하는 느낌}}
> 각 장마다 제목, 본문 카피(2-3줄), 시각 요소 설명을 포함하고,
> 후킹→공감→솔루션→CTA 흐름으로 1080×1080 기준으로 만들어줘.
> ```

- **체크리스트**:
  - [ ] 4장 구성 완료 (후킹→공감→솔루션→CTA)
  - [ ] 서비스 스크린샷 또는 목업 이미지 포함
  - [ ] 슬로건·컬러가 비주얼 컨셉과 일치
  - [ ] 마지막 장에 서비스 링크 또는 QR코드 포함

### 3. AI 쇼츠

- **목적**: AI 영상 생성 도구로 제품 홍보 영상 제작
- **포맷**: 9:16 세로, 30-60초
- **작성 가이드**:
  - AI 영상 생성 도구 활용 (Sora, Runway, Kling 등)
  - 장면 구성: 문제 상황 → 서비스 등장 → 해결 장면 → CTA
  - 자막 필수 (음소거 시청 대비)

| 장면 | 시간 | 화면 설명 | 자막 텍스트 | 전환 효과 |
|------|------|-----------|-------------|-----------|
| 후킹 | 0-3초 | | | |
| 문제 | 3-13초 | | | |
| 솔루션 | 13-28초 | | | |
| CTA | 28-33초 | | | |

- **사용 도구**: (Sora / Runway / Kling 등)

> **AI 프롬프트** -- AI 쇼츠 시나리오 생성
>
> ```
> @docs/01_plan.md 참고해서 AI 영상 생성 도구(Sora/Runway 등)용 쇼츠 시나리오를 만들어줘.
> 슬로건: {{선택한 슬로건}}
> 30-60초, 9:16 세로 포맷으로, 장면별 화면 설명/자막/전환 효과를 포함하고,
> 후킹(3초)→문제(10초)→솔루션(15초)→CTA(5초) 구성으로 만들어줘.
> 각 장면의 AI 영상 생성 프롬프트도 함께 작성해줘.
> ```

- **체크리스트**:
  - [ ] 30-60초 분량의 시나리오 완성
  - [ ] 각 장면별 자막 텍스트 작성
  - [ ] AI 영상 생성 도구 선정 및 생성 완료
  - [ ] 서비스 링크 또는 QR코드 마지막 장면에 포함

### 5. 블로그 포스트

- **목적**: 제품 아카이빙 및 SEO를 통한 검색 유입
- **포맷**: 네이버 블로그 1편 (이미지 5장 이상)
- **작성 가이드**:
  - 제목에 검색 키워드 포함 (예: "대학생 팀플 관리 앱")
  - 구성: 문제 제기 → 솔루션 소개 → 주요 기능 시연 캡처 → 사용 후기 → 링크
  - 카드뉴스 이미지와 시연 캡처를 본문에 활용

- **제목**:
- **본문 구성**:
  1. 문제 제기:
  2. 솔루션 소개:
  3. 기능별 시연 캡처:
  4. 사용 후기:
  5. CTA:
- **해시태그**:

> **AI 프롬프트** -- SEO 블로그 포스트 초안 생성
>
> ```
> @docs/01_plan.md 참고해서 네이버 블로그 SEO 최적화 포스트 초안을 작성해줘.
> 핵심 키워드: {{위에서 선정한 핵심 키워드}}
> 서비스 링크: {{배포된 URL}}
> 제목(30자 이내, 키워드 포함), 본문(문제→솔루션→기능 시연→후기→CTA),
> 이미지 삽입 위치 5곳, 해시태그 10개를 포함해줘.
> ```

- **체크리스트**:
  - [ ] 제목에 SEO 키워드 포함
  - [ ] 이미지 5장 이상 삽입 (캡처 + 카드뉴스)
  - [ ] 서비스 링크 포함
  - [ ] 해시태그 추가

## SNS 배포 가이드

| 플랫폼 | 올릴 콘텐츠 | 주의사항 | 완료 |
|--------|-------------|----------|------|
| 인스타그램 | 카드뉴스(피드) + AI 쇼츠·시연 쇼츠(릴스) + 스토리(피드/릴스 중 택1 공유) | 프로필 링크트리에 배포 사이트 연결 | |
| 네이버 블로그 | 블로그 포스트 (카드뉴스 이미지 + 영상 임베드 + 제품 소개) | SEO 키워드 제목 포함 | |
| 에브리타임 | 추천글 (제목 + 본문 3-5줄 + 링크) | 대학별 커뮤니티에 각각 게시 | |
| 유튜브 | AI 쇼츠 + 시연 쇼츠 + 전체 발표 영상(롱폼) | 쇼츠는 9:16 세로, 롱폼은 16:9 가로 | |

> **TIP**: 모든 플랫폼에 동시에 올리기보다, 인스타그램 → 에브리타임 → 블로그 → 유튜브 순서로 올리면 효율적입니다. 인스타 콘텐츠를 다른 플랫폼에 재활용할 수 있습니다.
